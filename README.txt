MoveMates Homepage
==================

So startest du lokal:
1. Ordner in VS Code öffnen.
2. index.html mit Live Server starten oder doppelklicken.
3. Bilder liegen in /assets.

Wichtige Anpassungen:
- index.html: Texte, Angebote, Telefonnummer, Mailadresse, Ort anpassen.
- assets/logo-card.jpg: später durch echtes transparentes Logo oder Hero-Bild ersetzen.
- impressum.html und datenschutz.html sind nur Platzhalter und müssen vor Veröffentlichung juristisch/inhaltlich korrekt ausgefüllt werden.

Das Formular ist bewusst als mailto-Demo gebaut. Für echte Website bitte ein Formular-Backend oder Buchungstool einbauen.
